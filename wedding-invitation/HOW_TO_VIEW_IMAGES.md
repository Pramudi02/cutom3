# 📸 How to View Your Lotus-Themed Images

## 🚀 View Everything Right Now

```bash
# 1. Navigate to project
cd wedding-invitation

# 2. Start development server
npm run dev

# 3. Open browser
# http://localhost:3000/rajitha-sayuri
```

**That's it!** Your invitation loads with all lotus images displayed.
